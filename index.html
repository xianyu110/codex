前往下载 Releases

选择适合您机器的可执行文件：

  • aarch64 → Apple M 系列芯片
  • x86_64 → Intel/AMD 64 位 CPU
  • darwin → macOS
  • pc-windows-msvc → Windows
  • unknown-linux-gnu/musl → Linux

以Apple M系列为例的安装命令：

# 下载指定版本 gh release download --repo openai/codex --pattern 'codex-aarch64-apple-darwin.tar.gz' # 解压文件 tar -xzf codex-aarch64-apple-darwin.tar.gz # 重命名 mv codex-aarch64-apple-darwin codex # 移动到系统路径 sudo mv codex /usr/local/bin/
